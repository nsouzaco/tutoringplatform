# Product Context

## Why This Exists
Online tutoring has grown significantly, but many platforms lack real-time collaboration tools and data-driven insights. This platform aims to:
- Simplify the connection between students and tutors
- Provide rich real-time learning experiences with video, chat, and collaborative notes
- Leverage AI to help tutors understand student engagement and learning outcomes
- Enable continuous improvement of teaching effectiveness

## Problems It Solves

### For Students
- **Discovery Challenge**: Finding available tutors with open time slots
- **Booking Friction**: Complex scheduling processes
- **Limited Interaction**: Need for real-time collaboration beyond just video
- **Accessibility**: Easy-to-use web platform without app downloads

### For Tutors
- **Availability Management**: Difficulty managing and displaying availability
- **Session Insights**: Lack of data-driven feedback on teaching effectiveness
- **Student Engagement**: Limited tools to assess student participation
- **Planning**: No structured way to track progress and plan future sessions

## How It Works

### Student Journey
1. **Discover** - Browse list of available tutors with profiles
2. **Schedule** - View tutor calendar and book available slots
3. **Prepare** - Receive confirmation and reminders
4. **Learn** - Join video session 5 min before start
5. **Engage** - Use video, chat, and view notes during session
6. **Complete** - Session ends automatically or manually

### Tutor Journey
1. **Setup** - Create profile and set availability (recurring or specific slots)
2. **Accept** - Students book into available time slots
3. **Prepare** - View upcoming sessions in dashboard
4. **Teach** - Start session, use video/chat/notes tools
5. **Document** - Take notes during session for later reference
6. **Analyze** - Review AI-generated report with engagement metrics and recommendations
7. **Improve** - Use insights to plan and improve future sessions

### AI Enhancement
After each session:
- Audio is transcribed (OpenAI Whisper)
- Chat history and notes are collected
- AI analyzes all data to generate structured report including:
  - Session summary
  - Topics covered
  - Questions asked
  - Student engagement level
  - Strengths and improvement areas
  - Recommendations and next steps

## User Experience Goals

### Simplicity
- Minimal steps to book and join sessions
- Clean, intuitive interfaces
- Clear status indicators

### Reliability
- Stable video conferencing
- Automatic session management
- Persistent data (chat, notes, reports)

### Insight
- Actionable AI reports for tutors
- Clear engagement metrics
- Structured recommendations

### Flexibility
- Multiple session duration options
- Recurring or one-time availability
- Manual or automatic session end

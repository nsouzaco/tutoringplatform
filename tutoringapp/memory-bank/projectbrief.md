# Project Brief: Tutoring Platform

## Project Name
Tutoring Platform (Name TBD)

## Vision
A web-based tutoring platform that connects students with tutors for one-on-one video sessions, enhanced with AI-powered insights to improve learning outcomes.

## Core Requirements

### User Types
1. **Students** - Browse tutors, book sessions, join video calls, participate in learning
2. **Tutors** - Set availability, manage sessions, facilitate learning, access AI-powered reports

### Essential Features
1. **Authentication & Onboarding**
   - Email/password authentication
   - Role selection (Student/Tutor)
   - Basic profile creation

2. **Tutor Availability Management**
   - Time slot specification (recurring or one-time)
   - Visual calendar view
   - Flexible session durations (15/30/45/60 min)
   - Timezone support

3. **Session Booking**
   - Browse available tutors
   - View tutor availability calendars
   - Book sessions with confirmation
   - Dashboard for managing bookings

4. **Video Sessions**
   - Real-time video conferencing (Jitsi Meet integration)
   - In-session text chat with timestamps
   - Collaborative note-taking
   - Automatic session timing and end notifications

5. **AI-Powered Post-Session Reports**
   - Automatic generation after session ends
   - Analysis of transcripts, chat, and notes
   - Structured report for tutors with engagement metrics and recommendations
   - Generated using OpenAI API (GPT-4/Whisper)

### Platform
Web-based MVP optimized for desktop and tablet

### Success Criteria
- Seamless booking and session experience
- Reliable video conferencing
- Accurate and useful AI reports
- Intuitive UX for both user types

## Out of Scope for MVP
- Payment processing
- Tutor verification
- Ratings/reviews
- Inter-session messaging
- Mobile apps
- Screen sharing
- Advanced calendar features
